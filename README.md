Training Bikilo
===============

Sencilla herramienta programada en PHP para analizar datos de entrenamiento de corredores.

La aplicación consulta la API del servicio [MyTrainik](https://mytrainik.com/), formatea los datos para su rápida lectura y calcula sumatorios y medias.

Los resultados se visualizan en una lista filtrable por fecha y tipo de entrenamiento.
